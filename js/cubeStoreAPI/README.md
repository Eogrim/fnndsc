# Cube Store API

Generates `es2015` bundle.

## Prerequisite

* yarn

## Commands

Install dependencies

``` bash
$> yarn install
```

Start tests in watch mode (used for developping)

``` bash
$> yarn start
```

Start test in single run mode (used by `Travis CI`)

``` bash
$> yarn test
```

Compile library to standalone bundle

``` bash
$> yarn build
```
